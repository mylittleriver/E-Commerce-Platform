# E-Commerce Platform

A simple **E-Commerce Platform** built with **Java, Spring Boot, MongoDB, Redis, and Docker**.  
Provides basic user authentication and product management APIs.

## Features

- User registration and login
- Add and list products
- Redis integration for caching
- Dockerized for easy deployment

## Tech Stack

- Java, Spring Boot
- MongoDB
- Redis
- Docker
- Maven

## Getting Started

### Prerequisites

- Java 17+
- Maven
- Docker
- MongoDB
- Redis

### Run Locally

```bash
git clone https://github.com/yourusername/ecommerce-platform.git
cd ecommerce-platform
mvn clean package
java -jar target/ecommerce-platform-0.0.1-SNAPSHOT.jar
