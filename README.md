# E-Commerce Platform

A simple **E-Commerce Platform** built with **Java, Spring Boot, MongoDB, Redis, and Docker**.  
Provides basic user authentication and product management APIs.

## Features

- User registration and login
- Add and list products
- Redis integration for caching
- Dockerized for easy deployment

## Tech Stack

- Java, Spring Boot
- MongoDB
- Redis
- Docker
- Maven

## Getting Started

### Prerequisites

- Java 17+
- Maven
- Docker
- MongoDB
- Redis

### Run Locally

```bash
git clone https://github.com/mylittleriver/E-Commerce-Platform.git
cd E-Commerce-Platform
mvn clean package
java -jar target/E-Commerce-Platform-0.0.1-SNAPSHOT.jar
